# icinga2-plugin-raspberrypi_voltage
Measuring voltage values and check whether host is throttled or not 
* Tested on Raspberry Pi Models: 1, 2 & 3
* Status **worksforme**

### Usage
```
$ ./check_raspberrypi_voltage
```

### Example
```
$ ./check_raspberrypi_voltage
OK: all seems fine.

```
